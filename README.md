# 球球大作战：大逃杀

一个单人版 Canvas 小游戏。玩家控制球球吃豆成长，在缩小的安全区里和 AI 球球对抗，使用分身、吐球、炸刺等机制活到最后。

## 玩法

- 鼠标移动 / 手机触摸：控制球球移动方向
- Space：分身，最多 16 个分身
- W：吐球
- R：重新开始
- 手机按钮：右下角分身、吐球

## 规则

- 大球可以吞噬明显更小的球。
- 吃彩色小豆会增加质量。
- 吐球会消耗自身质量并向前发射质量球。
- 大球吃到绿色刺球会炸成多个分身。
- 橙色安全区会持续缩小，圈外会掉质量。
- 剩下最后一个阵营时结算胜负。

## 本地运行

```powershell
npm.cmd run serve
```

然后打开：

```text
http://localhost:5173/
```

## 测试

```powershell
npm.cmd test
```

测试覆盖分身上限、吐球消耗、炸刺分裂、吞噬碰撞和安全区伤害。
